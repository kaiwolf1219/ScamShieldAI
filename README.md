# ScamShield AI

ScamShield AI is a cybersecurity-focused web application being developed for the SANS Institute Find Evil Hackathon.

The goal of ScamShield AI is to help users identify potentially malicious emails, text messages, and social engineering attempts by analyzing message content and highlighting common scam indicators.

## Features (Planned)

* Analyze emails, text messages, and direct messages
* Generate a scam risk score
* Identify suspicious language and social engineering tactics
* Explain why a message was flagged
* Provide recommendations for staying safe online

## Tech Stack

### Backend

* SQLite

### Machine Learning (Planned)

* Scikit-learn
* Custom phishing/scam detection classifier

## Motivation

Scammers increasingly use phishing, impersonation, and social engineering attacks to target individuals online. ScamShield AI aims to provide an accessible tool that helps users recognize these threats before they become victims.
